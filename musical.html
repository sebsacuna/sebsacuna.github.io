Musical

Musical is an algorithm created by K. Agarwal and R. Machan for reconstruction of nanoscopy images, allowing super-resolution.

Currently I'm working on 2 implemntations of the algorithm. The first is written in Java as a ImageJ plugin.

The second correspond to a Python package. With both the objective is to achieve a speed that allows real-time application.
